## 2. ゲーム概要

プレイヤーは丸石で囲まれたバトルフィールド内で、ランダムに出現する黄色の彩釉テラコッタを剣で破壊してスコアを競うアクションゲーム。時間経過とともにゾンビと葉ブロックが増加し、難易度が段階的に上昇する。

| 項目 | 内容 |
|------|------|
| ジャンル | アクション / サバイバル系ミニゲーム |
| プレイ人数 | 1人〜（マルチプレイヤー対応可） |
| ゲームモード | サバイバルモード（ブロック破壊のため） |
| ワールド種別 | フラットワールド、オーバーワールドのみ使用 |

## 3. バトルフィールド仕様

### 3.1 サイズと位置

| 項目 | 値 |
|------|-----|
| 横幅（X方向） | 30ブロック |
| 奥行（Z方向） | 30ブロック |
| 壁の高さ | 10ブロック |
| 中心座標 | X=0, Z=0 |
| 地面の高さ | Y=-61 ※フラットワールドの地面に合わせて調整 |

### 3.2 構造

- 外周は丸石の壁（4面）で囲まれている
- 内部（29×29ブロック）はプレイエリアとして空洞になっている
- 天井はない（上部は開放）

### 3.3 構築時の処理

1. バトルフィールド内部を空気で満たし、前回のゲームの残存物をすべて除去する
2. 外周4面に丸石の壁を構築する

---

## 4. ゲームの開始とリセット

### 4.1 ゲーム開始方法

- プレイヤーが任意のタイミングでゲームを開始できる手段を用意する（例：チャットコマンド、ボタン等）
- ワールドロード直後に自動開始はしない（プレイヤーの準備を待つため）

### 4.2 初期化タイミング

- ゲーム開始が指示されてから、**約5秒間の待機**を設ける（ワールドの完全読み込みを待つため）

### 4.3 リセット（再スタート）

- ゲーム中いつでも同じ手段でリセットできること
- リセットを実行すると、ゲームが初期状態に完全に戻り、最初から遊び直せること
- 何度リセットしても同じ初期状態が再現されること

### 4.4 初期化で行うこと

| # | 内容 |
|---|------|
| 1 | バトルフィールド周辺のプレイヤー以外の全エンティティを除去する |
| 2 | スコア表示をサイドバーに設定する（表示名：「Level」） |
| 3 | 全プレイヤーのスコアを0にする |
| 4 | 全プレイヤーにダイヤモンドの剣×1と土ブロック×64を付与する |
| 5 | 全プレイヤーをバトルフィールド内の開始地点に移動する（バトルフィールド中心からX-3, Z-3の位置） |
| 6 | 画面にゲーム開始メッセージ「テラコッタを破壊せよ！」を表示する |
| 7 | バトルフィールドを構築する（3章参照） |

## 5. ゲームの進行ルール

### 5.1 テラコッタの出現

- ゲーム開始後、バトルフィールド内のランダムな位置に黄色の彩釉テラコッタが1つ出現する
- 出現時に「新しいテラコッタが出現した！」とメッセージを表示する
- テラコッタは地面から1ブロックの高さに配置される
- 同時に存在するテラコッタは常に1つだけ

### 5.2 テラコッタの破壊判定

- 約1秒ごとにテラコッタの存在を確認する
- テラコッタがあった場所にテラコッタが存在しなくなった場合、「破壊された」と判定する

### 5.3 破壊後の流れ

1. スコアが1加算される
2. サイドバーのスコア表示が更新される
3. 「テラコッタを破壊した！数秒後に新しいテラコッタが出現するぞ。」とメッセージを表示する
4. **約2秒の待機後**、次のテラコッタがランダムな位置に出現する

---

## 6. 難易度の上昇

### 6.1 敵モブ（ゾンビ）の出現

- 一定間隔でバトルフィールド内のランダムな位置にゾンビが1〜2体出現する
- スコアが上がるほど出現間隔が短くなる

| スコア | 出現間隔の目安 |
|--------|--------------|
| 0 | 約30秒 |
| 2 | 約10秒 |
| 5 | 約5秒 |
| 9 | 約3秒 |

#### 出現間隔の計算式

```
出現間隔（秒） = ceil(200 / ((スコア + 1) / 3)) / 20
```

### 6.2 障害物（葉ブロック）の出現

- 約1.5秒ごとに、バトルフィールド内のランダムな位置に葉ブロックが10個配置される
- 高さは地面から0〜9ブロックの範囲でランダム
- 時間経過とともにバトルフィールド内が葉で埋まっていく

#### 障害物の役割

- プレイヤーの視界を遮り、テラコッタの発見を困難にする
- 移動経路を妨害する
- ゲームが長引くほどバトルフィールド内が混沌とし、難易度が上がる

## 7. スコアリング

| 項目 | 内容 |
|------|------|
| スコアの名称 | Level |
| 表示位置 | 画面サイドバー |
| 初期値 | 0 |
| 加算条件 | テラコッタを1つ破壊するごとに+1 |
| 影響 | スコアが上がるほどゾンビの出現頻度が増加する |

---

## 8. ゲームの終了

### 8.1 終了条件

- 全プレイヤーが倒される（体力が0になる）とゲーム終了

### 8.2 終了時の処理

- 最終スコアを画面に表示する
- 「ゲーム終了！スコア：○○」のようなメッセージを表示する

### 8.3 終了後

- リセットを実行することで最初から遊び直せる
